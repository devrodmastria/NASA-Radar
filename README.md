# Asteroid Radar App
* Android Kotlin App
* Made for Udacity nanodegree - project 2

## Description
This project consists of two screens:
1. Main screen
   1. Image of the day
   2. List of near-earth-orbit asteroids
2. Detail screen
   1. Image of asteroid
   2. scientific description
